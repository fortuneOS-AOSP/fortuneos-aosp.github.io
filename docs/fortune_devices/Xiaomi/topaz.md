## Installation Guide
:::caution
Make sure to backup your data before proceeding.
:::

### Installing ROM
1. Download the latest release of Fortune.
2. Reboot into recovery mode.
3. Perform a Format data.
4. Flash custom rom
6. Reflash Recovery
7. Reboot System
